🍷 Viña Cascas - Gestión de Vinos (API)

API RESTful desarrollada con Java 17 y Spring Boot, usando MySQL para gestionar de manera eficiente compras, ventas e inventario de vinos de Viña Cascas.

⸻

🚀 Funcionalidades principales
	•	📦 Gestión de productos y stock
	•	🛒 Registro de compras y ventas
	•	📊 Control de inventario con actualizaciones automáticas
	•	📄 Generación de reportes básicos de transacciones
	•	🔒 Endpoints seguros listos para integración con frontend

⸻

🛠 Tecnologías utilizadas
	•	Java 17
	•	Spring Boot
	•	Spring Data JPA
	•	MySQL
	•	Maven / Gradle
	•	REST API

⸻

⚡ Características adicionales
	•	Estructura de proyecto escalable y modular
	•	Soporte para futuras integraciones con frontend en Angular o React
	•	Buenas prácticas de desarrollo y manejo de excepciones
